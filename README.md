# Sistema Predictor de Salario para Primer Empleo

Proyecto desarrollado en Go que utiliza técnicas de clasificación para estimar el salario esperado de una persona en su primer empleo, considerando factores académicos, habilidades profesionales y experiencia previa.

## Características
- Implementado en Go.
- Uso de archivos CSV para almacenar datos.
- Predicción del salario esperado.
- Análisis de características académicas y profesionales.
- Clasificación de rangos salariales mediante inteligencia artificial.

## Archivos del proyecto
- main.go: Código principal.
- salarios.csv: Base de datos de entrenamiento.
- modelo.nb: Modelo entrenado.
- go.mod: Dependencias del proyecto.

## Ejecución

```bash
go run main.go
```
